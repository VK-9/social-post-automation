# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately by opening a security advisory at:

https://github.com/VK-9/social-post-automation/security/advisories/new

Do **not** open a public issue for security vulnerabilities.

## What to Include

- A clear description of the vulnerability
- Steps to reproduce (if applicable)
- Potential impact
- Any suggested fixes (optional)

## Response

I'll acknowledge receipt within 48 hours and aim to provide a fix or mitigation within 7 days.

## Scope

This project is a hobby/educational tool. Supabase, OpenRouter, Cloudflare, Postproxy, and JSON2Video are third-party services — vulnerabilities in those services should be reported to their respective teams.
